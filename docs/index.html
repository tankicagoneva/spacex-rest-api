Launchpads

Get all launchpads

Retrieve a list of launchpads, optionally filtered by status.

query Parameters
status
string
Default: "active"
Enum: "active" "retired" "under-construction"

The status of the launchpad (e.g., 'active', 'inactive').

Responses

Development server

http://localhost:3000/launchpads

Create a new launchpad

Request Body schema: application/json
required
id
required
string

Unique identifier for the launchpad

name
required
string

Short name of the launchpad

full_name
required
string

Full official name of the launchpad

locality
string or null

Locality of the launchpad

region
Array of strings

Region(s) where the launchpad is located

latitude
number or null <double>

Latitude coordinate

longitude
number or null <double>

Longitude coordinate

launch_attempts
integer or null

Total number of launch attempts from this launchpad

launch_successes
integer or null

Total number of successful launches

rockets
Array of strings

List of rocket IDs associated with the launchpad

timezone
string or null

Timezone of the launchpad

status
string or null

Current status of the launchpad (e.g., active, retired)

details
string or null

Additional details about the launchpad

object or null

JSON object containing images related to the launchpad

launches
Array of strings

List of launch IDs from this launchpad

Responses

Development server

http://localhost:3000/closest

required

Longitude of the location (between -180 and 180).
